
## Spotify Clone 🎵

A full-stack Spotify Clone built using React JS, Tailwind CSS, Node.js, Express.js, MongoDB, and Cloudinary.
This project allows users to stream songs, browse albums, and manage music through an admin dashboard.

## Demo

- User Panel: https://spotify-full-stack-frontend-clone.onrender.com
- Admin Panel: https://spotify-full-stack-admin-3w5n.onrender.com

## Features

- Play and Pause Songs  
- Browse Music Albums  
- Online Music Streaming  
- Upload Songs and Album Images  
- Cloudinary Media Storage  
- MongoDB Database Integration  
- Admin Dashboard for Music Management  
- Add New Songs and Albums  
- Delete Songs and Albums  
- Fast and Responsive User Interface  
- Modern Design using Tailwind CSS  
- Fully Responsive for Mobile and Desktop  
- Built with React JS and Vite  
- Full Stack Deployment on Render  
- REST API Integration using Express.js  
- Organized MERN Stack Project Structure
